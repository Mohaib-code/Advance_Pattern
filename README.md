# Java Pattern Practice

This repository contains multiple pattern printing programs written in **Java**. The goal of this project is to practice **nested loops, conditional statements, and pattern logic** commonly asked in programming exercises, coding interviews, and academic coursework.

## Patterns Implemented

- Hollow Rectangle Pattern
- Inverted Star Pyramid
- Inverted Number Half Pyramid
- Floyd's Triangle
- 0–1 Triangle Pattern
- Butterfly Pattern
- Solid Rhombus Pattern
- Hollow Rhombus Pattern
- Diamond Pattern

## Concepts Used

- Nested loops
- Conditional statements
- Pattern logic
- Basic Java methods

## Purpose

This project is useful for beginners learning **Java programming** and **loop-based problem solving**. It helps improve logical thinking and understanding of how nested loops work to generate different patterns.
